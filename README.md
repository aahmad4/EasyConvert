## Setup

#### Clone
```bash
git clone https://github.com/aahmad4/EasyConvert
```

#### Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the required packages.
```bash
pip install -r requirements.txt
```

#### Usage
```
cd EasyConvert
```
```
python main.py
```
